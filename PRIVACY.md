# Privacy Policy

Effective date: 2026-05-23

Discord Search Blocker does not collect, store, sell, share, or transmit personal data.

The extension runs only on Discord channel pages and hides the visible in-channel search field in the browser. It does not read Discord messages, account information, usernames, server data, channel data, browsing history, or search queries.

Discord Search Blocker has no analytics, no tracking, no advertising, no remote server, and no external network requests.

The extension's host access is used only so its content script can run on `discord.com/channels/*` and apply the visible UI change.

If this privacy policy changes, the updated version will be published in this repository.
