# ko

## Short Description

Discord 채널 안의 검색 입력란을 숨깁니다.

## Full Description

Discord Search Blocker는 Discord 웹 앱에서 검색 입력란을 제거합니다.

음소거했거나 차단했거나 의도적으로 피하려는 Discord 채널에 더 쉽게 접근하지 못하게 하고 싶을 때 사용하세요. Discord 검색은 다시 보고 싶지 않은 곳의 메시지를 여전히 보여줄 수 있습니다. 이 확장 프로그램은 그 진입점을 숨깁니다.

기능:
- discord.com의 채널 내 검색 입력란 숨김
- Discord 웹 앱에서만 실행
- 설치 후 조용히 작동
- 메시지를 읽지 않고, 데이터를 수집하지 않으며, 분석 정보를 보내지 않음
