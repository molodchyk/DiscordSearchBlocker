# bn

## Short Description

Discord চ্যানেলের ভেতরের সার্চ ক্ষেত্র লুকিয়ে রাখে।

## Full Description

Discord Search Blocker Discord-এর ওয়েব অ্যাপ থেকে সার্চ ক্ষেত্র সরিয়ে দেয়।

আপনি যখন মিউট, ব্লক করা বা ইচ্ছাকৃতভাবে এড়িয়ে চলা Discord চ্যানেলগুলোকে সহজে খুঁজে না পেতে চান, তখন এটি ব্যবহার করুন। Discord সার্চ এমন জায়গার বার্তাও দেখাতে পারে যেখানে আপনি আর ফিরতে চান না; এই এক্সটেনশন সেই সার্চ প্রবেশপথটি লুকিয়ে রাখে।

এটি যা করে:
- discord.com-এ চ্যানেলের ভেতরের সার্চ ক্ষেত্র লুকায়
- শুধু Discord ওয়েব অ্যাপে চলে
- ইনস্টলেশনের পর নীরবে কাজ করে
- বার্তা পড়ে না, ডেটা সংগ্রহ করে না, অ্যানালিটিক্স পাঠায় না
