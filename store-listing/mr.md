# mr

## Short Description

Discord चॅनेलमधील शोध फील्ड लपवते.

## Full Description

Discord Search Blocker Discord च्या वेब अॅपमधून शोध फील्ड काढून टाकते.

म्यूट, ब्लॉक केलेली किंवा जाणीवपूर्वक टाळलेली Discord चॅनेल्स सहज उपलब्ध होऊ नयेत असे वाटत असल्यास हे वापरा. Discord शोध तुम्ही पुन्हा पाहू इच्छित नसलेल्या ठिकाणांतील संदेश अजूनही दाखवू शकतो; हे एक्स्टेंशन तो प्रवेशबिंदू लपवते.

हे काय करते:
- discord.com वर चॅनेलमधील शोध फील्ड लपवते
- फक्त Discord वेब अॅपवर चालते
- इंस्टॉल केल्यानंतर शांतपणे काम करते
- संदेश वाचत नाही, डेटा गोळा करत नाही किंवा अॅनालिटिक्स पाठवत नाही
